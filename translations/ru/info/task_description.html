Ты разрабатываешь игру, в которой фишки двигаются по периметру квадратной карты (как в Монополии)

Требуется написать функцию, которая возвращает значения координат фишки после того, как ей "сходили". Координаты фишки - кортеж, содержащий номера строки и столбца. Координаты нижнего левого угла - (0, 0).

    Функция принимает 3 целочисленных аргумента:
  • первый - длина стороны игровой карты;
  • второй - номер клетки исходной позиции фишки;

    • Нумерация клеток по периметру карты начинается с 0 в правом нижнем углу и осуществляется по часовой стрелке.

  • третий - количество "шагов" фишки;

    • A Положительное количество "шагов" говорит о том, что фишкой нужно идти по часовой стрелке, отрицательное - против часовой.

Example:

{% if interpreter.slug == "js-node" %} 
square_board(4, 1, 4) == [1, 0]
square_board(6, 2, -3) == [4, 5]
{% else %} 
square_board(4, 1, 4) == (1, 0)
square_board(6, 2, -3) == (4, 5)
{% endif %}

Входные данные: {% if interpreter.slug == "js-node" %} {% else %} 3 целочисленных аргумента: длина стороны игровой карты, текущая позиция фишки, число "шагов". {% endif %}

Выходные данные: Координаты фишки после хода (кортеж, содержащий номер строки и номер столбца).

Как это используется:
Используется для вычисления координат объекта, движущегося по периметру.

Предусловия:

  • 3 ≤ The length of the side ≤ 11
  • 0 ≤ The position of the token < (side-1)*4
